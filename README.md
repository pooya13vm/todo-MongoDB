# todo-app
## General info
It as a training project to work on MongoDB Database

## Technologies
> Project is created with:
- [x] node.js
- [x] express.js
- [x] MongoDB
- [x] mongoose ODM
- [x] ejs (template engine)
- [x] Bootstrap 5
- [x] font-awesome
***
> the project is made with MVC design pattern

## Setup
To run this project, install it locally using npm:
- $ npm install
- $ npm start
